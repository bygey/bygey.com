1.2被控变量的选择

1. 应当尽量选择质量指标参数作为被控变量.
2. 无法实现1时,应当选择一个与产品质量指标有单值对应关系的时间间接指标参数作为被控变量.
3. 间接指标参数要有足够大的灵敏度,以便反应产品质量的变化.
4. 被控变量的选择要考虑工艺的合理性和仪表现状.

控制变量的选择

应能够较大程度影响被控参数.一般可以调整物料量或能量参数.
