# 控制通道特性对控制质量的影响

| 特性参数       | 对静态质量的影响     | 对动态质量的影响              |
| -------------- | -------------------- | ----------------------------- |
| $$K_0$$增加    | 余差减小(稳定前提下) | 系统趋于振荡                  |
| $$T_0$$增加    | 无影响               | 过渡过程时间增加,系统频率变慢 |
| $$\tau_0$$增加 | 无影响               | 稳定程度大大降低              |
